# SnapCart AI

Building **SAM (Shopping Agent Model)** — domain-specific foundation models for commerce reasoning.

We believe shopping AI should understand products the way expert shoppers do: reading between the lines of reviews, tracking price patterns, and making genuinely personalized recommendations.

## Open Source

| Repository | Description | License |
|---|---|---|
| [sam-bench](https://github.com/snapcart-ai/sam-bench) | Benchmark for evaluating AI shopping assistants — 1,200 tasks, 8 types, 3 difficulty levels | Apache 2.0 |
| [sam-api](https://github.com/snapcart-ai/sam-api) | FastAPI server for SAM-Bench — tasks, leaderboard, evaluation, and inference | AGPL-3.0 |
| [sam-sdk](https://github.com/snapcart-ai/sam-sdk) | Typed Python client for the SAM API (sync + async) | MIT |
| [research](https://github.com/snapcart-ai/research) | Research papers (SAM-Bench, SAM-1, Commerce Evaluation Stack) | CC BY 4.0 |
